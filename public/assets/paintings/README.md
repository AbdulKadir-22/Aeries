# Adding Your Own Paintings

1. Place your painting images here in the `public/assets/paintings/` folder (e.g., `my_art.jpg`, `water_lily.png`).
2. Open `src/data/paintings.json`.
3. Update the `image` field strings to point to your new files: `"/assets/paintings/my_art.jpg"`.
4. Adjust the title, category, and description as preferred.
